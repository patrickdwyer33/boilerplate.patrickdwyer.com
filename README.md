# boilerplate.patrickdwyer.com

Boilerplate repository for personal projects. Comes with basic PWA functionality.

## Instructions

1. `git clone git@github.com:patrickdwyer33/boilerplate.patrickdwyer.com.git {directory-name}`
2. `git remote remove origin`
3. Set up GitHub repository for project
3. Update project name in `package.json`
4. Update manifest metadata in `vite.config.js`
5. Update relevant variables in `deploy.sh`
